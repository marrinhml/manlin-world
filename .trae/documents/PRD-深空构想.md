## 1. 产品概述

蛮林世界（原深空构想）是一个科幻创意分享与社交平台，专门用来记录、展示和讨论科幻点子、科幻概念及科幻随笔。面向科幻爱好者、写作者和创意工作者，提供一个沉浸式的个人与轻社交自留地。

- **核心痛点**：科幻爱好者缺少一个同时具备创作、社交和沉浸体验的科幻主题平台
- **目标用户**：科幻创作者、科幻爱好者、脑洞收藏者、文艺科幻读者
- **运营状态**：已上线，部署于 Vercel，使用 Supabase 作为后端

## 2. 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 访客 | 直接访问网页 | 浏览内容、搜索、阅读详情 |
| 注册用户 | 邮箱注册登录 | 发布内容、点赞、评论、收藏、好友系统、编辑/删除自己的内容 |

## 3. 核心功能模块

### 3.1 引导体验

| 功能 | 说明 |
|------|------|
| 全屏启动动画 | 科幻终端风格引导页，含扫描线、全息投影、旋转光环等视觉特效 |
| 加载进度动画 | SVG 环形进度条，3.8 秒 easeOutCubic 动画模拟系统加载 |
| 背景音乐启动 | 进入后自动播放背景音乐（BGM，1 小时循环），提升沉浸感 |
| 关于弹窗 | 首次进入自动弹出站点介绍 |

### 3.2 内容系统

| 功能 | 说明 |
|------|------|
| 发布内容 | 支持 **3 种分类**：科幻点子（idea）、科幻概念（concept）、科幻随笔（essay） |
| 富文本编辑 | contenteditable 编辑器，支持文字样式、图片嵌入（base64，上限 500KB） |
| 标签系统 | 发布时可添加多个标签，支持标签筛选 |
| Markdown 渲染 | 使用 marked.js 渲染 Markdown 格式内容 |
| 全文详情 | 模态弹窗展示完整内容，含作者信息、分类、标签、时间 |
| 分类筛选 | 按全部/点子/概念/随笔切换 |
| 标签过滤 | 从所有内容中提取标签生成过滤栏 |
| 关键词搜索 | 实时搜索标题和正文 |
| 排序 | 按发布时间 / 热度（点赞+评论数）排序 |
| 分页 | 每页 15 条内容，支持翻页 |

### 3.3 互动系统

| 功能 | 说明 |
|------|------|
| 点赞 | 支持三级点赞：内容点赞、评论点赞、回复点赞 |
| 评论 | 发表评论、展开/收起评论区 |
| 回复 | 对评论进行回复 |
| 收藏 | 收藏内容，可筛选只看收藏列表 |
| 分享 | 复制内容链接到剪贴板 |
| 浏览计数 | 查看评论时自动增加浏览次数 |

### 3.4 用户系统

| 功能 | 说明 |
|------|------|
| 注册 | 邮箱 + 昵称 + 密码，随机科幻风昵称生成（30 个预置昵称池） |
| 登录 | 邮箱 + 密码 |
| 会话恢复 | 页面刷新自动恢复登录状态 |
| 个人档案 | 展示头像、昵称、邮箱、注册日期、5 项统计指标（发布/获赞/浏览/评论/收藏） |
| 他人档案 | 查看其他用户头像、昵称、统计、发布列表 |
| 设置 | 修改昵称、修改密码（需验证旧密码） |
| 头像系统 | 上传头像（含裁剪：拖拽 + 缩放）、随机生成 SVG 头像、prefers-color-scheme 自适应 |
| 账号注销 | 确认流程后提示联系管理员 |

### 3.5 社交系统

| 功能 | 说明 |
|------|------|
| 好友搜索 | 按邮箱或昵称搜索已注册用户 |
| 好友请求 | 发送请求、接受请求、拒绝请求 |
| 好友关系 | 好友列表管理、删除好友 |
| 请求红点 | 有待处理请求时显示 badge |

### 3.6 主题与个性化

| 功能 | 说明 |
|------|------|
| 明暗主题 | 亮色/暗色双主题，点击浮动按钮切换 |
| 主题持久化 | localStorage 保存主题选择，刷新不丢失 |

### 3.7 资讯模块

| 功能 | 说明 |
|------|------|
| 星际资讯 | 聚合 RSS 科幻资讯，通过 Vercel Serverless Function 获取 |
| 分类过滤 | 全部 / 快闻 / 赛事 |
| 增量刷新 | 获取最新资讯并去重合并 |
| 离线缓存 | 首次失败时降级到 localStorage 缓存 |
| 分页 | 资讯列表分页显示 |
| 更新时间 | 显示最后更新时间 |

### 3.8 站点特性

| 功能 | 说明 |
|------|------|
| 背景音乐 | 浮动音乐控制器，播放/暂停，进度条，localStorage 持久化播放状态和进度位置 |
| 网络状态 | 在线/离线检测，Supabase 连接状态指示器 |
| Toast 通知 | 成功/失败两种样式通知，自动消失 |
| 卡片入场动画 | IntersectionObserver 实现滚动淡入动画 |
| 响应式设计 | 自适应桌面/平板/手机，600px 断点切换布局 |
| 星空背景 | box-shadow 绘制的星点背景（桌面端带闪烁动画，移动端降级为静态） |

## 4. 用户流程

```
访客访问
  → 全屏引导页
  → 点击「开始探测任务」
  → 加载进度动画（3.8s）
  → 进入主站 + 背景音乐自动播放
  → 弹出关于弹窗

浏览流程：
  → 浏览内容卡片列表
  → 筛选分类 / 搜索关键词 / 过滤标签 / 切换排序
  → 点击卡片进入全文详情
  → 点赞 / 收藏 / 评论 / 回复

创作流程（需登录）：
  → 点击「发布新内容」
  → 填写标题 + 正文（富文本）+ 标签 + 选择分类
  → 发布 → 内容即时出现在列表

社交流程（需登录）：
  → 好友弹窗 → 搜索用户 → 发送请求
  → 对方接受 → 建立好友关系
  → 查看他人档案 → 可添加/删除好友
```

## 5. 设计规范

- **主色调**：深空黑 (#0a0e17) + 星云浅蓝 (#7ec8e3) + 微弱金光 (#c9a84c)
- **辅助色**：深蓝灰 (#141b2b)、浅灰文字 (#b8c6d4)、发光边框色 (rgba(126, 200, 227, 0.3))
- **英文排版**：JetBrains Mono（等宽字体）
- **中文排版**：Noto Sans SC
- **断点**：桌面 > 600px，移动端 ≤ 599px

## 6. 性能目标

| 指标 | 目标 |
|------|------|
| 首屏加载 | < 3s（包含 Google Fonts 异步加载） |
| 交互响应 | 点击事件 < 100ms |
| 列表渲染 | 100 条内容 < 500ms |
| 动画帧率 | 60fps（桌面），移动端禁用重型 animation/backdrop-filter |
